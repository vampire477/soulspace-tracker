# Soulspace Tracker

A modern web application for tracking and managing your personal soul space journey.

## Overview

Soulspace Tracker is a TypeScript-based application designed to help users monitor, track, and optimize their personal wellness and spiritual journey. Built with modern web technologies, it provides an intuitive and responsive interface for seamless user experience.

**Live Application**: [https://soulspace-tracker.lovable.app](https://soulspace-tracker.lovable.app)

## Technology Stack

This project leverages a contemporary, performant tech stack:

- **Build Tool**: [Vite](https://vitejs.dev/) - Lightning-fast frontend tooling
- **Language**: [TypeScript](https://www.typescriptlang.org/) - Type-safe JavaScript
- **Framework**: [React](https://react.dev/) - Composable UI library
- **UI Components**: [shadcn-ui](https://ui.shadcn.com/) - High-quality, accessible components
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework

## Features

- Intuitive dashboard for tracking progress
- Real-time data visualization
- Responsive design for desktop and mobile devices
- Type-safe codebase with TypeScript

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/vampire477/soulspace-tracker.git
   cd soulspace-tracker
   ```

2. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server
   ```bash
   npm run dev
   # or
   yarn dev
   ```

The application will be available at `http://localhost:5173`

## Build

To create a production build:

```bash
npm run build
# or
yarn build
```
